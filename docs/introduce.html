介绍

一套企业级的前端设计语言和基于 React 的实现。

特性

  • 企业级金融产品的交互语言和视觉体系。
  • 丰富实用的 React UI 组件。
  • 基于 React 的组件化开发模式。
  • 背靠 npm 生态圈。
  • 基于 webpack 的调试构建方案，支持 ES6。

示例

var antd = require('antd');
var Datepicker = antd.Datepicker;

React.render(<Datepicker />, mountNode);
