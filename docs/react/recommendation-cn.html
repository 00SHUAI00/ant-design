社区精选组件

antd 是 Ant Design 设计规范的 React 实现，所以我们倾向于只提供符合该规范、且带有视觉展现的 UI 组件，也尽量不重复造轮子。我们推荐使用以下社区已有的优秀实现，与 antd 形成互补：

  • 路由： react-router

  • 布局： react-blocks

  • 拖拽： react-dnd

  • 编辑器： react-codemirror

  • 拾色器： rc-color-picker、react-color

  • Media Query： react-responsive、react-media

  • 复制到剪贴板： react-copy-to-clipboard

  • 管理 document head： react-helmet

  • Font Awesome 图标: https://github.com/andreypopp/react-fa
