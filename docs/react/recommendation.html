Third-Party Libraries

antd is designed to provide high-quality React UI components which following Ant Design Specification. So, we are not going to implement other functions which are not relative to antd's target, and recommend to use excellent third-party library which came from React community:

  • Router: react-router

  • Layout: react-blocks

  • Drag and drop: react-dnd

  • Code Editor: react-codemirror

  • Rich Text Editor: react-quill react-draft-wysiwyg

  • Color Picker: rc-color-picker, react-color

  • Media Query: react-responsive, react-media

  • Copy to clipboard: react-copy-to-clipboard

  • Document head manager: react-helmet

  • Font Awesome Icons: react-fa

  • QR Code: : qrcode.react
