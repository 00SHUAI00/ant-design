Carousel

A carousel component. Scales with its container.

When To Use#

  • When there is a group of content on the same level.

  • When there is insufficient content space, it can be used to save space in the form of a resolving door.

  • Commonly used for a group of pictures/cards.

Basic

Basic usage.

import { Carousel } from 'antd';

function onChange(a, b, c) {
  console.log(a, b, c);
}

ReactDOM.render(
  <Carousel afterChange={onChange}>
    <div><h3>1</h3></div>
    <div><h3>2</h3></div>
    <div><h3>3</h3></div>
    <div><h3>4</h3></div>
  </Carousel>
, mountNode);

Scroll automatically

Timing of scrolling to the next card/picture.

import { Carousel } from 'antd';

ReactDOM.render(
  <Carousel autoplay>
    <div><h3>1</h3></div>
    <div><h3>2</h3></div>
    <div><h3>3</h3></div>
    <div><h3>4</h3></div>
  </Carousel>
, mountNode);

API#

PropertyDescriptionTypeDefault
effectAnimation effect, either scrollx or fadeStringscrollx
dotsWhether to show the dots at the bottom of the galleryBooleantrue
verticalWhether to use a vertical displayBooleanfalse
autoplayWhether to scroll automaticallyBooleanfalse
easingTransition nameStringlinear
beforeChangeCallback function called before the current index changesfunction(from, to)
afterChangeCallback function called after the current index changesfunction(current)

For more info on the parameters, refer to the https://github.com/akiran/react-slick
